Гайды Dota 2

Шаг 1: Включить консоль и бинды
▼
1.1 Открой Настройки -> вкладку "Дополнительно".
1.2 Включи "Консоль разработчика". Клавиша для открытия консоли в игре — `~` (тильда).

1.3 Чтобы быстро пикать героя без мышки, вставь в консоль эту команду:
bind "F1" "dota_hero_pick"
Шаг 2: Настройки Nvidia для Доты
▼
2.1 Открой Панель управления Nvidia -> "Управление параметрами 3D".
2.2 Перейди в "Программные настройки", выбери Dota 2 из списка.

2.3 Выставь эти параметры для максимального FPS:
• Режим управления электропитанием: Предпочитать максимальную производительность
• Фильтрация текстур (качество): Высокопроизводительная
• Сглаживание (FXAA): Выкл
• Максимальная частота кадров: Отключить
Шаг 3: Создать autoexec.cfg
▼
Чтобы настройки сохранялись после перезапуска игры, создай файл autoexec.cfg.

3.1 Открой папку с игрой по пути:
C:\Steam\steamapps\common\dota 2 beta\game\dota\cfg

3.2 Создай там файл autoexec.cfg и вставь в него команды для автоматической загрузки.

Пример команд для вставки:
dota_camera_distance 1800 fps_max 0 bind "F1" "dota_hero_pick"
